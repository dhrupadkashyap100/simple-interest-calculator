# Simple Interest Calculator

This project is a Simple Interest Calculator that allows users to compute simple interest based on three parameters: principal amount, rate of interest, and time period.

## Features
- Calculate simple interest using the formula:  
  **Simple Interest (SI) = (Principal × Rate × Time) / 100**
- User-friendly interface to input values and view results.
- Validation to ensure proper input values.

## Installation
1. Clone the repository:  
   `git clone https://github.com/dhrupadkashyap100/simple-interest-calculator.git`
2. Navigate to the project directory:  
   `cd simple-interest-calculator`
3. Open the index.html file in your web browser.

## Usage
1. Enter the principal amount, rate of interest, and time period in the respective fields.
2. Click the 'Calculate' button to get your simple interest.
